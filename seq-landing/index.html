Arquitectura Modular

Dos Modulos, Una Plataforma

SeQ combina capacidades de escaneo de vulnerabilidades con gestion segura de secretos en una solucion unificada.

Operativo

Sentinel

REST API para escaneo de vulnerabilidades

  • Autenticacion OAuth 2.0 con tokens JWT y refresh tokens
  • Escaneo de puertos con Nmap (soporte CIDR)
  • Analisis web con Nikto (configs inseguras, headers)
  • Vulnerabilidades OpenVAS con scoring CVSS
  • Generacion de reportes PDF con patron Strategy
En Desarrollo

Acheron

Sistema de gestion cifrada de secretos via Vaults

  • Vault cifrado para objetos sensibles (Account, CreditCard)
  • Control de acceso mediante ACL compartido
  • IDs compuestos unicos (codigo tipo + secuencial)
  • Autenticacion OAuth 2.0 unificada con Sentinel
  • Apps: AcheronCore (Java), Mobile (Kotlin), Web

Ejemplos de API

Interfaz REST Simple

API RESTful intuitiva para integracion rapida con tus sistemas existentes.

Autenticacion
POST 
# Obtener token de acceso
POST /api/auth/login

{
  "username": "admin",
  "password": "secure_password"
}

# Response
{
  "access_token": "eyJhbGciOiJIUzI1NiIs...",
  "refresh_token": "eyJhbGciOiJIUzI1NiIs...",
  "token_type": "Bearer",
  "expires_in": 3600
}
Escaneo Nmap
POST 
# Iniciar escaneo de puertos
POST /api/scan/nmap
Authorization: Bearer <token>

{
  "targets": "192.168.1.0/24",
  "ports": "22,80,443,8080",
  "scan_type": "tcp_connect"
}

# Response
{
  "scan_id": "sc_7f8a9b2c",
  "status": "running",
  "started_at": "2024-01-15T10:30:00Z"
}
Analisis OpenVAS
POST 
# Escaneo completo de vulnerabilidades
POST /api/scan/openvas
Authorization: Bearer <token>

{
  "target": "10.0.0.5",
  "config": "full_deep",
  "schedule": "now"
}

# Configs: full_fast, full_deep, full_ultimate
Consultar Resultados
GET 
# Obtener resultados del escaneo
GET /api/scan/results/{scan_id}
Authorization: Bearer <token>

# Response con vulnerabilidades
{
  "scan_id": "sc_7f8a9b2c",
  "status": "completed",
  "findings": [
    {
      "severity": "high",
      "cvss": 8.5,
      "cve": "CVE-2024-1234"
    }
  ]
}

Instalacion Rapida

Pon SeQ en funcionamiento en minutos con estos simples pasos.

1
Clonar el repositorio
git clone https://github.com/gamustea/SeQ.git && cd SeQ
2
Instalar dependencias
pip install -r requirements.txt
3
Configurar variables de entorno
cp .env.example .env && nano .env
4
Iniciar el servidor
flask run --host=0.0.0.0 --port=5000
