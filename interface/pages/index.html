Capacidades del Sistema

Arquitectura modular basada en Flask y Managers de escaneo.

⚡

Nmap Scanner

Gestión asíncrona de escaneos de puertos. Define tus objetivos mediante headers HTTP estándar.

Headers:
host: "192.168.1.10"
ports: "80,443,8080"
🕷️

Nikto Integration

Detección de vulnerabilidades web. Soporta configuración de timeout y targets específicos.

POST /scans/nikto/start
Header 'target': "webapp.local"
Query 'timeout': 180
📄

Smart Reporting

Motor PDFCreator integrado con estrategias de impresión personalizadas para cada tipo de escaneo.

GET /scans/generate-pdf-base64
Strategies: Nmap & Nikto
Format: Base64 / Stream
