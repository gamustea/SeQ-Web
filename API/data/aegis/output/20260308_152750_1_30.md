# Píldora de concienciación:
**Drive-by download: infección al navegar**

Cada vez que navegas por internet, estás expuesto a un tipo de ataque conocido como drive-by download. Este método de infección aprovecha vulnerabilidades en tu navegador o software para introducir malware en tu dispositivo sin que tú lo notes. Dado que la empresa EMESA utiliza tecnologías como HPE y Sonicwall, es esencial estar al tanto de este riesgo para proteger tanto tus datos como los de la compañía.

Aquí te presentamos algunas medidas concretas que puedes tomar para minimizar el riesgo de una infección por drive-by download:

- **Instala y actualiza extensiones de seguridad confiables.** Estas extensiones pueden bloquear sitios web sospechosos antes de que puedas caer en una trampa. Considera usar [uBlock Origin](https://github.com/gorhill/uBlock) o [NoScript](https://noscript.net/) para navegar con más seguridad.
- **Evita sitios web dudosos.** Las páginas con mucha publicidad, descargas gratuitas de software o que prometen ganancias fáciles son especialmente riesgosas. Si un sitio web no parece seguro o es desconocido, es mejor no entrar.
- **Configura tu navegador para bloquear activamente el contenido malicioso.** Los ajustes de seguridad de tu navegador pueden ayudarte a prevenir el acceso a sitios web peligrosos. Por ejemplo, en Google Chrome, activa las políticas de seguridad extendidas para bloquear contenido malicioso y scripts potencialmente dañinos.
- **Mantén actualizado tu sistema operativo y software.** Las actualizaciones no solo mejoran el rendimiento, sino que también cierran las brechas que los atacantes podrían aprovechar. Asegúrate de instalar las actualizaciones tan pronto como sean disponibles.
- **No hagas clic en enlaces dudosos.** Si recibes un enlace por correo electrónico o en redes sociales que no reconozcas, es mejor no hacer clic en él. Los enlaces maliciosos pueden llevar a sitios web infectados.
- **Utiliza un firewall robusto.** Software como Sonicwall, utilizado por EMESA, puede ayudarte a bloquear tráfico no deseado y a prevenir el acceso a sitios web potencialmente peligrosos.
- **Ejercita la cautela con los anuncios.** Muchos ataques de drive-by download se originan a través de anuncios en sitios web no seguros. Desactiva los anuncios o usa bloqueadores de anuncios para minimizar este riesgo.

Si tienes dudas sobre cómo protegerte contra los ataques de drive-by download, no dudes en contactar con ciberseguridad@emesa.com para obtener más orientación. Mantenerse informado y tomar medidas preventivas es crucial para mantener la seguridad de la empresa.
