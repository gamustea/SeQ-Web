**Navega seguro por Internet**

Tu forma de navegar por Internet puede marcar la diferencia entre
mantener segura la información de la empresa o exponerla a riesgos
innecesarios. Un descuido al acceder a sitios web o descargar archivos
puede abrir puertas a ciberdelincuentes. Aquí tienes algunos consejos
para navegar de forma segura por Internet:

-   Evita conectarte a redes Wi-Fi públicas sin protección cuando
    trabajes con información corporativa. Las redes de cafeterías,
    aeropuertos u hoteles son fáciles de monitorizar por atacantes que
    pueden interceptar tus datos.

-   Mantén tu navegador actualizado. Las actualizaciones corrigen
    vulnerabilidades de seguridad que los ciberdelincuentes pueden
    explotar para acceder a tu equipo o robar información sensible.

-   Verifica siempre que las páginas web usen conexión segura
    (https://). Fíjate en el candado que aparece en la barra de
    direcciones antes de introducir cualquier información personal o
    corporativa.

-   Ten cuidado con las URLs. Los atacantes crean sitios web falsos muy
    parecidos a los legítimos para robar credenciales. Revisa bien la
    dirección completa antes de iniciar sesión en cualquier servicio.

-   No descargues archivos de sitios web no confiables. Muchos ataques
    de malware comienzan con la descarga de un archivo aparentemente
    inofensivo desde una página sospechosa.

-   Usa extensiones de navegador con precaución. Solo instala
    complementos de fuentes oficiales y revisa los permisos que
    solicitan. Algunas extensiones maliciosas pueden capturar todo lo
    que escribes, incluyendo contraseñas.

-   Cierra las sesiones cuando termines de trabajar, especialmente si
    usas un equipo compartido. Así evitarás que otras personas accedan a
    tus cuentas o información corporativa.

-   Si algo te resulta sospechoso mientras navegas (ventanas emergentes
    extrañas, avisos de seguridad, comportamiento inusual), no sigas
    adelante. Cierra el navegador y repórtalo al departamento de IT.
