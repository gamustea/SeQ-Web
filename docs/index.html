Funcionalidades

Capacidades principales de la plataforma

Escaneo de Puertos

Descubre puertos abiertos en hosts o rangos CIDR usando Nmap con soporte multi-host.

Análisis Web

Detecta configuraciones inseguras y rutas expuestas en servidores HTTP/HTTPS con Nikto.

Vulnerability Assessment

Escaneos completos con OpenVAS/GVM y puntuaciones CVSS para cada hallazgo.

Informes con IA

Generación de PDF con análisis ejecutivo, nivel de riesgo y recomendaciones por IA local (Ollama).

OAuth 2.0 + JWT

Sistema de autenticación moderno con refresh tokens y control de sesiones por usuario.

Píldoras de Concienciación

Contenido educativo generado por IA adaptado al contexto de cada organización.

Alertas de Vulnerabilidades

Feed automático de CVEs recientes desde INCIBE-CERT y CIRCL/NVD.

Vault Cifrado

Gestión segura de cuentas y tarjetas de crédito con cifrado AES-256.

Docker

Infraestructura completa con PostgreSQL, OpenVAS y Ollama containerizados.
